# Retail Sales Analysis

## 專案說明
使用 Python 與 SQL 分析 Superstore 零售銷售資料，
探討各地區、品項的銷售額、利潤及折扣率之趨勢。

## 使用技術
- Python
- pandas
- matplotlib
- SQL（SQLite）
- SQLAlchemy

## 分析內容
1. 資料概覽
2. 地區銷售分析
3. 品項銷售分析
4. 地區 + 品項交叉分析
5. 虧損分析
6. 折扣率與虧損分析

## 主要發現
- West 地區整體表現最強，South 地區最弱
- Furniture 品項利潤極低，需檢討銷售策略
- 折扣超過 40% 後利潤普遍轉為負值
- Tables 商品即使無折扣利潤仍偏低

## 資料來源
Kaggle - Sample Superstore Dataset
